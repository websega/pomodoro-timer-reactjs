## Pomodoro timer.
Помидорковый таймер на ReactJs.

##### Scripts
- Install packages    ```npm i```
- Start dev-server    ```npm run start```
- Development build   ```npm run dev```
- Production build    ```npm run build```
- Сode check          ```npm run lint```
- Check and fix       ```npm run lint:fix```
